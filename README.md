# News-source

A news aggregation and display application.

## Getting Started

### Prerequisites
- Git installed on your system
- A GitHub account

### Initial Setup

If you're starting a new repository, follow these steps:

1. Navigate to your project directory:
```bash
cd "path/to/your/project"
```

2. Check the status of your repository:
```bash
git status
```

3. View recent commits:
```bash
git log --oneline -n 5
```

4. Rename your branch to main (if needed):
```bash
git branch -M main
```

5. Add the remote origin:
```bash
git remote add origin https://github.com/<your-username>/<repo>.git
```

6. Push your code to the main branch:
```bash
git push -u origin main
```

### Cloning an Existing Repository

If you're cloning this repository:

```bash
git clone https://github.com/tharsh1111-ops/News-source.git
cd News-source
```
